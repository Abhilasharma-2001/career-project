# 🚀 System Monitoring Dashboard

A real-time system monitoring dashboard built using Flask.

## 📸 Features

* 📈 Live CPU graph
* 💾 Memory & Disk monitoring
* ⚠️ Alert system for high CPU usage
* 🔄 Real-time updates (no page reload)

## 🛠 Tech Stack

* Python
* Flask
* psutil
* Chart.js

## ▶️ How to Run

```bash
pip install -r requirements.txt
python3 dashboard.py
```

Then open: http://127.0.0.1:5000
